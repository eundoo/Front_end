# 문자열

자바스크립트는 글자 하나만 저장할 수 있는 별도의 자료형이 없다.

## 문자열의 불변성
---
문자열은 수정할 수 없다. (중간에 한글자를 수정한다던지..) 그래서 이런 문제를 피하려면 완전히 새로운 문자열을 하나 만들어서 할당해야 한다.

### 
```javascript
    (`${}`)     //글자랑 표현식을 함께 삽입
    //특수문자 줄바꿈.. 홈페이지 참조
    .length     //특정글자에 접근
    toLowerCase
    indexOf     //부분문자열 찾기
    
    includes
    startsWith
    endsWith

    .slice      //부분문자열 추출

    //문자열은 유니코드대로 비교
```
